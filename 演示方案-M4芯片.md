## 🎯 推荐方案: 使用localtunnel (更简单)

ngrok下载有问题,我推荐用**localtunnel**,更简单且不需要注册:

### 安装和使用localtunnel:

```bash
# 1. 安装localtunnel (需要Node.js)
npm install -g localtunnel

# 如果没有Node.js,先安装:
# 访问 https://nodejs.org 下载Mac ARM版本

# 2. 启动服务
cd "/Users/sunsensen/claude code/智能数据平台/backend"
source venv/bin/activate
python app.py

# 3. 新开终端,创建公网访问:
lt --port 8000

# 4. 会显示一个网址,类似:
# your url is: https://random-name.loca.lt
# 复制这个地址发给客户!
```

## 🌐 方案4: 手动下载ngrok (如果你有下载工具)

由于curl下载ngrok失败,你可以:

### 步骤1: 浏览器下载
访问这个链接(用浏览器,不是终端):
```
https://ngrok.com/download
```

### 步骤2: 选择Mac ARM版本
下载页面会自动识别你的M4芯片

### 步骤3: 安装
```bash
# 下载后解压,得到ngrok文件

# 移动到系统路径
sudo mv ~/Downloads/ngrok /usr/local/bin/

# 添加执行权限
chmod +x /usr/local/bin/ngrok

# 验证
ngrok version

# 启动
cd "/Users/sunsensen/claude code/智能数据平台/backend"
source venv/bin/activate
python app.py  # 先启动后端

# 新开终端
ngrok http 8000  # 启动ngrok
```
