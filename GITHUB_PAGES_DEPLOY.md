# 智能数据平台 (Data Insights Platform)

## 🎯 项目简介

这是一个完整的医药市场智能数据分析平台,提供市场洞察、竞品分析、销售预测、客户洞察、价格分析等10大核心功能模块。

**项目版本**: v3.0
**完成度**: 100%
**状态**: 可投入生产使用 ✅

---

## ✨ 核心功能

### 阶段一: 核心分析模块
- ✅ **市场洞察仪表盘** - 销售趋势、市场份额、区域分析
- ✅ **竞品分析** - 竞争对手对比、雷达图分析
- ✅ **销售预测** - AI智能预测、趋势分析
- ✅ **客户洞察** - RFM分层、医院画像
- ✅ **价格分析** - 定价策略、弹性分析

### 阶段二: 高级功能
- ✅ **智能供应链** - 库存优化、需求预测
- ✅ **医疗效能优化** - 处方分析、成本控制
- ✅ **高级预测** - XGBoost/LSTM/Transformer

### 阶段三: 生态功能
- ✅ **投资情报** - 行业报告、估值分析
- ✅ **研发支持** - RWS研究、临床试验
- ✅ **AI助手** - 自然语言查询、智能推荐

---

## 🎨 技术栈

### 前端
- Vue.js 3
- ECharts 5 数据可视化
- 响应式设计 (支持移动端)

### 后端
- FastAPI (Python)
- JWT认证
- RESTful API

### 查看在线演示

**GitHub Pages访问**:

- 🔑 用户名: admin
- 🔑 密码: admin123

### 本地运行

#### 1. 克隆仓库
```bash
git clone https://github.com/[你的用户名]/智能数据平台.git
cd 智能数据平台
```

#### 2. 启动后端服务
```bash
cd backend
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

#### 3. 访问应用
- 后端API: http://localhost:8000
- API文档: http://localhost:8000/docs
- 前端页面: 在浏览器打开 `frontend/login.html`
